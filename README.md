# IoT Phase 4 - Descriptive Data Analysis

## 📊 Project Description
This project performs a descriptive analysis of environmental IoT sensor data, including temperature, humidity, and gas measurements, in order to identify patterns and environmental conditions.

## 📁 Dataset
IoT Telemetry Dataset (Kaggle)

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## 📈 Analysis Performed
- Data loading and exploration
- Data cleaning (nulls and duplicates check)
- Descriptive statistics analysis
- Data visualization (temperature, humidity, correlation)
- Pattern identification

## 🔍 Key Findings
- Temperature remains stable around 22°C
- Humidity shows high variability
- Gas levels remain low and safe
- No strong correlation between main variables

## 📌 Conclusion
IoT sensors provide effective real-time monitoring of environmental conditions, allowing detection of variations and supporting decision-making in urban environments.
